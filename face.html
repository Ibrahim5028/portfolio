Facial Recognition Project

About the Project


I developed a Raspberry Pi project that controlled a door lock using facial recognition. The system, using a camera connected to the Pi, captured images, compared them with a stored database, and unlocked the door upon recognizing an authorized person. The lock mechanism, made with gears and locks, was created using 3D printing and CAD. Servo motors, controlled by electronic circuit boards, were attached to the door lock, activating it for identified individuals.


For this system, I used Python libraries like OpenCV for image processing, dlib and face_recognition for facial detection, RPi.GPIO for interfacing with the Raspberry Pi hardware, and numpy for supporting mathematical operations. This project merged secure facial recognition with the adaptability of Raspberry Pi for automated access control.
