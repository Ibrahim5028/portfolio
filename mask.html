COVID Mask Project

About the Project


My CAD course project had the goal of utilizing CAD to build a COVID mask that was durable, 3D printable, cheap, and easy to assemble. It was a good educational resource and great introduction to 3D CAD modeling that also employed a real-world impactful use.


As this was my first-time using CAD this project had helped me understand concepts such as sketching irregular shaped geometries, extrusion, and part assembly.


Skill Acquired:


CAD Software Proficiency, Design for Manufacturability, Sketching Irregular Shapes, Extrusion Techniques, Part Assembly
