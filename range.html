Range Detector Project

About the Project


In our instrumentation course project, we developed a range detector system utilizing block programming and electronic circuitry. I proposed creating a device that emits a sound when reaching a pre-set distance, applicable in scenarios like car collision sensors, warehouse inventory management, and construction for precise distance measurements.


The system's hardware included an ultrasonic sensor for distance detection, an Arduino board for control and programming, and an infrared sensor for motion detection. This multifaceted approach allowed us to create a versatile and practical range detection tool.


Skill Acquired:


Block Programming, Electronic Circuitry Design, Arduino Programming and Use, Problem-Solving with Technology
